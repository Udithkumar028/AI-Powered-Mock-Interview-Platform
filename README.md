# 🎯 AI-Powered Mock Interview Platform

An intelligent AI-powered interview preparation platform designed to help students, freshers, and professionals practice interviews, improve communication skills, and receive personalized feedback in real-time.

The platform simulates real interview experiences using Artificial Intelligence, helping users build confidence and improve performance before attending actual interviews. AI-powered interview platforms commonly focus on personalized questions, feedback, and performance analysis. :contentReference[oaicite:0]{index=0}

## ⚡ Installation

### Clone Repository

```bash
git clone https://github.com/Udithkumar028/AI-Powered-Mock-Interview-Platform.git

cd AI-Powered-Mock-Interview-Platform
```

### Install Frontend Dependencies

```bash
cd client
npm install
```

### Install Backend Dependencies

```bash
cd ../server
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000

MONGODB_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET_KEY

GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

---

## ▶️ Run Project

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm run dev
```

Application will run on:

```bash
Frontend: http://localhost:5173

Backend: http://localhost:5000
```
